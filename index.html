neuropipe by ntblab

NeuroPipe is a framework for reproducible fMRI analysis projects with FSL, developed in the Turk-Browne Lab at Princeton University.

Install

See the tutorial: http://github.com/ntblab/neuropipe-support/blob/master/doc/tutorial/tutorial.rst

$ git clone git://github.com/ntblab/neuropipe
